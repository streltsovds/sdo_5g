Для развертывания PHP Code Sniffer необходимо скопировать файл pre-commit в директорию .git/hooks и настроить в нём константы:

PHP_DIR              - директория, в которой расположен php.exe
PHP_INI_DIR          - директория, в которой лежит рабочий php.ini
VALIDATE_ONLY_SYNTAX - если 1, то проверяется только синтаксис php-файлов

После этого при всех коммитах будет производиться проверка оформления кода + проверка синтаксиса php-файлов
