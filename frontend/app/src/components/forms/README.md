# Компоненты форм

Большинство компонентов из этой папки реализуют представление php-класса `HM_Form_Element_Vue_Element`. У таких компонентов в `props` есть поле `attribs` .

Демо: /demo/vue/form, компонент `demo-hm-form-components`.
